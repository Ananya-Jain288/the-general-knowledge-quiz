🧠 General Knowledge Quiz

📌 Project Overview:-
The General Knowledge Quiz is a Python-based console application developed as part of the DecodeLabs Python Programming Internship.
The application allows users to answer multiple-choice questions, calculates the score, displays the final result, and stores quiz records.

🚀 Features:-
* Multiple Choice Questions
* Instant Answer Validation
* Final Score Calculation
* Percentage Display
* Pass/Fail Result
* Save Score History
* View Previous Scores

🛠 Technologies Used:-
* Python
* Lists
* Dictionaries
* Functions
* Loops
* Conditional Statements
* File Handling

▶️ How to Run:-
1. Install Python.
2. Download or clone this repository.
3. Open the project folder.
4. Run:
thegenknowledgequiz.py

* Decision making using if-else
* User input handling
* Working with loops
