🧠 The General Knowledge Quiz

📌 Project Overview:-
The General Knowledge Quiz Application is a Python-based console application developed as part of the DecodeLabs Python Programming Internship.
The application allows users to answer multiple-choice questions, calculates the final score, and provides performance feedback.

🚀 Features:-
->Multiple-choice quiz
->Easy, Medium, and Hard levels
->Automatic score calculation
->Percentage calculation
->Performance feedback
->Random question order
->Replay quiz
->Save results to a file

🛠 Technologies Used:-
->Python
->Functions
->Lists
->Dictionaries
->Loops
->Conditional Statements
->Random Module
->File Handling

▶️ How to Run:-
->Install Python 3.x
->Download or clone this repository
->Open the project folder
->Run:
thegenknowledgequiz.py

📚 Learning Outcomes:-
->Decision-making using if-else
->Data handling using lists and dictionaries
->Working with Python modules
->User input validation
->Score calculation and result analysis
